Se quiser que continue a contar, devemos escrever "start" na segunda <ol> tag (onde queremos que continue) e escrever o número onde terminou a contagem passada.

Lista de definições

Listas de definições servem como dicionários, escrevemos os termos e as suas definições.
a tag praslistas de definições é <dl>, utilizamos <dt> para especificar o termo e <dd> para a descrição.
EX.:

HTML
Linguagem de marcação de texto para a escrita
CSS
linguagem de marcação para criação do Design
JavaScript
Linguagem de programação para a criação de interatividade pro website
