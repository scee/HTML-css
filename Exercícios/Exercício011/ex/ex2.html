Em HTML5, podemos inserir os seguintes elementos:

  • Parágrafos
    Usar a Tag <P
  • Títulos
    Usar a Tag <H1>, <H2>, <H3>, etc
  • Imagens
    Usar a Tag <img>
  • ABreviações
    Usar a Tag <abbr>
  • Códigos-fonte
    Usar a Tag <code>
  • Listas numeradas
    Usar a Tag <ol> <li>
  • Listas não numeradas
    Usar a Tag <ul> <li>
