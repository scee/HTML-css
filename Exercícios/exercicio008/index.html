Principais formatações

Negrito/Destaque

Nesta frase temos uma palavra em negrito. Para negritar a palavra usamos a tag <b> (Forma) que é B de bold

Nesta frase temos uma palavra em negrito. Para negritar a palavra, também podemos usar a tag <strong> (Semântica) que é a que mais vamos usar

Itálico ou ênfase

Nesta frase temos duas palavras em itálico. Para deixar assim a palavra, usamos a tag <i> (forma)

Nesta frase temos duas palavras em itálico. Para deixar assim a palavra, usamos a tag <em> (semântica para "Ênfase") Também será a nossa principal escolha

Texto marcado

Podemos criar também textos marcados utilizando a tag <mark>. Olha... este aqui está marcado

Se crio outra fase e uso a tag <mark>, a cor n vem igual.
Para que seja igual, tenho de cria uma tag style na área do cabeçalho uma tag style. Seria maiso ou menos assim:
<style>
mark { background-color: lime (por exemplo);}
</style>

Texto grande / Texto pequeno

Aqui temos um texto grande.
Aqui temos um texto pequeno. Sendo que para texto grande é <big> e para pequeno é <small> (easy assim mesmo)

Texto excluído e texto Inserido

Podemos marcar o texto como excluído (para indicar que deve ser lido mas não considerado) utilizando a tag <del>
Podemos marcar o texto como inserido (dar ênfase e indicar que foi adicionado depois) utilizando a tag <ins> ou mesmo <u> (mas é melhor utilizar mesmo a "ins")

Texto sobreescrito / Texto subscrito

Ex de texto sobreescrito: X elevado a 20 + 3. A tag: <sup> resolve, e com isso temos : X20+3

Ex de texto subescrito: fórmula quimica da água .A tag: <sub> e fica H2O
